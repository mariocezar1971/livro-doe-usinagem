# Dados experimentais do livro

## Conteúdo planejado

- `ligas_caracterizacao.csv` — propriedades mecânicas das 6 ligas (R, Ar, Hd)
- `ensaios_2k.csv` — resultados do planejamento fatorial 2^k
- `ensaios_pcc.csv` — resultados do Planejamento Composto Central
- `regressao_global.csv` — dados consolidados para regressão global I/II

## Formato

Todos os arquivos em UTF-8, separador vírgula, com cabeçalho.
