# Códigos do livro

## Estrutura

- `R/` — códigos R por capítulo (FrF2, rsm, desirability, ggplot2)
- `python/` — códigos Python por capítulo (pymoo, pyDOE3, matplotlib)

## Reproduzibilidade

Todos os códigos foram testados com:
- R 4.4 ou superior
- Python 3.11 ou superior

Para instalar dependências:

```bash
# R
Rscript ../setup/setup-r-env.R

# Python
pip install -r ../setup/requirements.txt
```
